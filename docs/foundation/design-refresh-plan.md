# デザイン刷新・パターン検証計画

> 目的：りんごちゃん薬局の信頼性・地域性・相談しやすさを保ちながら、前衛的で印象に残るコーポレートサイト表現を複数パターンで検証する。
> 作成日：2026-07-18

---

## 1. 前提と守ること

- 技術スタックは既存方針どおり **Astro + Vanilla CSS** を維持する。
- Tailwind / UIフレームワーク / ページビルダーは導入しない。
- 薬局サイトであるため、前衛性を出しても以下は崩さない。
  - 高齢者にも読める文字サイズ・行間・コントラスト
  - 電話・LINE・アクセスなど重要導線の分かりやすさ
  - 薬機法・YMYL領域にふさわしい誇張しない文言
  - 地域の調剤薬局としての安心感
- 今後も複数パターンを試すため、ページを直接作り込む前に **共通コンポーネントとデザイントークン** を整備する。

## 2. 参考サイトから抽出する方向性

### 2.0 薬局専門制作事例（PHARMA STUDIO 制作実績）から見る堅実な要件

PHARMA STUDIO の薬局ホームページ制作実績一覧を参照し、薬局サイトでは前衛性よりも先に以下の「迷わない情報設計」が重視されやすいと整理する。

- **来局前の不安解消**：営業時間、所在地、電話、LINE/予約、対応サービスをトップ付近で確認できる。
- **地域の信頼感**：薬局外観、スタッフ、待合室など、実在感のある写真で「ここに行けばよい」と分かる。
- **サービス理解の速さ**：処方箋受付、在宅、健康相談、OTC、ジェネリックなどをカードや短文で並べ、探す負担を減らす。
- **採用・会社情報への安心感**：代表挨拶、理念、店舗情報、運営会社を整えて、患者・求職者・医療機関の信頼を得る。
- **装飾より可読性**：派手な演出より、余白、やさしい色、読みやすい見出し、押しやすいボタンを優先する。
- **スマホ前提の導線**：地図アプリ、電話、LINE、処方箋送信など、行動に直結するボタンを分かりやすく置く。

この観察から、りんごちゃん薬局の刷新でも「おしゃれさ」は情報の見つけやすさを上書きしない。前衛的な案を試す場合も、薬局サイトとしての必須情報は常に固定モジュール化して守る。

### 2.1 道の駅ウェルネスあらお

- 参考要素
  - 大きな写真・イラスト・余白で地域性を印象づけるトップ構成
  - 「About Us」「Products」「News」など、英字ラベルと日本語見出しを併用する編集感
  - セクションごとに背景や写真配置を変え、スクロールするほど表情が変わる構成
- 薬局サイトへの応用
  - 「荒川・町屋の暮らしに近い薬局」という地域性を、写真・短いコピー・カード群で見せる
  - 商品ページは「取扱商品 Products」として、薬局らしい情報整理と雑誌的な見せ方を両立する

### 2.2 L'OCCITANE Sustainability

- 参考要素
  - 画面冒頭の強いキャンペーンビジュアル
  - スクロール誘導、数値、短い宣言文によるストーリー設計
  - ブランドカラーを大胆に使い、商品より先に思想や活動を見せる構成
- 薬局サイトへの応用
  - 「相談できる」「待ち時間を減らせる」「地域の健康を支える」などを数値・ステップ・宣言文で見せる
  - 薬局の清潔感を崩さない範囲で、赤・緑・白を大胆に使う

### 2.3 星街すいせい Official Site

- 参考要素
  - 固定的な世界観、強いタイポグラフィ、カテゴリごとの情報一覧
  - NEWS / LIVE / DISCOGRAPHY のように情報量が多い領域をリズム良く配置
  - ブランド名・英字・余白・ビジュアルで「個性」を先に伝える構成
- 薬局サイトへの応用
  - 「RINGOCHAN PHARMACY」「MACHIYA / ARAKAWA」などの大きな英字タイポを背景要素として使う
  - お知らせ・商品・営業時間・アクセスを、情報整理されたモジュールとして並べる

## 3. ブラッシュアップ案（まず8パターン）

### Pattern A：Local Editorial / 地域編集型

- キーワード：地域、余白、やわらかい写真、雑誌風、親しみ
- 参考寄り：ウェルネスあらお
- 向いている目的：初見の安心感、町屋・荒川の地域密着感
- トップ構成案
  1. 大きな写真ヒーロー + 「町屋の暮らしに寄り添う薬局」
  2. `About`：薬局の考え方を短文で紹介
  3. `Products`：商品カードを横スクロールまたは3列で紹介
  4. `Consultation`：LINE・処方箋受付・市販薬相談を3カード化
  5. `Access`：営業時間と地図を大きめに配置
- 視覚要素
  - 生成り白、淡いグリーン、アクセント赤を少量
  - 手書き風ではなく、上品な丸み・余白で親しみを出す

### Pattern B：Wellness Brand / ウェルネスブランド型

- キーワード：清潔、サステナブル、宣言、数値、ブランド感
- 参考寄り：L'OCCITANE Sustainability
- 向いている目的：薬局を「健康を支えるブランド」として見せる
- トップ構成案
  1. フルスクリーン寄りヒーロー + 強いブランドコピー
  2. `OUR PROMISE`：3つの約束（待たない、相談できる、地域を支える）
  3. `FLOW`：LINE受付・来局・受け取りの流れ
  4. `PRODUCTS`：市販薬・健康食品をカテゴリ別に見せる
  5. `DATA`：駅徒歩5分、土日営業、LINE受付などを数字で表示
- 視覚要素
  - 白背景に大胆な赤/緑の面構成
  - 円形・波形・大きな番号・スクロール誘導

### Pattern C：Neo Corporate / 前衛コーポレート型

- キーワード：タイポグラフィ、黒/白/赤/緑、余白、グリッド、動き
- 参考寄り：星街すいせい Official Site
- 向いている目的：他の薬局サイトと明確に差別化した印象づくり
- トップ構成案
  1. 大きな英字ロゴタイポ + 薬局写真の切り抜き配置
  2. 固定サイドラベルまたは縦書きラベルで「ARAKAWA / MACHIYA」
  3. `TOPICS`：お知らせ・商品・SNSをタブ状に整理
  4. `SERVICES`：処方箋・ジェネリック・市販薬相談を大きなカードで表示
  5. `CONTACT`：LINE CTA を強いビジュアルブロックにする
- 視覚要素
  - 余白を広く、文字を大きく、カードはシャープに
  - 動きは控えめなフェード・スライド程度（高齢者配慮）


### Pattern D：Clinic Minimal / 静かな医療ミニマル型

- キーワード：白、余白、罫線、整然、安心、上質
- 参考寄り：医療機関・研究機関の清潔なコーポレートサイト
- 向いている目的：信頼性を最優先しつつ、古い薬局サイト感をなくす
- トップ構成案
  1. 白背景 + 小さな薬局写真 + 端正なコピー
  2. `SERVICE INDEX`：処方箋、LINE受付、ジェネリック、市販薬相談を細い罫線カードで整理
  3. `PRODUCTS`：商品は写真を大きくせず、情報の正確さを重視して一覧化
  4. `INFORMATION`：営業時間・電話・住所を一画面で把握できる設計
- 視覚要素
  - 影を控え、罫線と余白で構造を作る
  - 赤はCTAのみに限定し、医療サイトとしての落ち着きを担保する

### Pattern E：Ringo Pop / りんごポップ型

- キーワード：りんご、丸み、明るさ、親しみ、家族、かわいいが子どもっぽくしすぎない
- 参考寄り：地域ブランド・ファミリー向けサービスサイト
- 向いている目的：名前の覚えやすさと相談しやすさを最大化する
- トップ構成案
  1. りんごモチーフの円形グラフィック + 店舗写真
  2. `3つの相談`：処方箋、市販薬、健康相談を丸カードで表示
  3. `PRODUCTS`：商品カードにカテゴリ別の淡い背景色を付ける
  4. `SNS / NEWS`：Instagramやお知らせをやわらかいタイルで見せる
- 視覚要素
  - 角丸、円、淡い赤、淡い緑を多めに使う
  - 文字は読みやすさ重視で、装飾は背景グラフィックに寄せる

### Pattern F：Pharmacy Magazine / 薬局マガジン型

- キーワード：編集、読み物、商品紹介、健康コラム、余白、写真キャプション
- 参考寄り：ライフスタイルメディア・ブランドジャーナル
- 向いている目的：将来的に商品紹介や健康情報を増やす前提のSEO強化
- トップ構成案
  1. `FEATURE`：季節の相談・おすすめ商品を大きな記事カードで表示
  2. `PRODUCTS`：カテゴリ別に商品カードを並べる
  3. `GUIDE`：LINE受付や薬の相談方法を読み物風に案内
  4. `SHOP INFO`：店舗情報を末尾に整理
- 視覚要素
  - 見出し、本文、キャプションの階層を明確にする
  - microCMSで商品だけでなく将来コラムを持つ場合にも拡張しやすい

### Pattern G：Trust First Pharmacy / 信頼最優先型

- キーワード：営業時間、アクセス、電話、LINE、外観写真、迷わない
- 参考寄り：薬局専門制作事例の堅実な情報設計
- 向いている目的：初回来局者・高齢者・処方箋利用者に最短で必要情報を届ける
- トップ構成案
  1. 薬局外観 + 「町屋駅徒歩5分」「土日営業」「LINE受付」をファーストビューに固定表示
  2. `FIRST ACTION`：電話・LINE・地図の3ボタンを大きく配置
  3. `SERVICES`：処方箋受付、市販薬相談、ジェネリック、健康相談を説明付きカード化
  4. `SHOP INFO`：営業時間・定休日・住所・地図を1セクションで完結
- 視覚要素
  - デザインより行動導線を優先する
  - 写真は外観・入口・待合室など「行く前に安心できるもの」を中心に使う

### Pattern H：Community Care / 地域医療連携型

- キーワード：かかりつけ、在宅、医療機関連携、相談、地域包括ケア
- 参考寄り：地域薬局・かかりつけ薬局の堅実なコーポレートサイト
- 向いている目的：患者だけでなく、家族・介護者・医療機関からの信頼も獲得する
- トップ構成案
  1. 「地域の健康相談窓口」というメッセージを中心に置く
  2. `CARE MENU`：服薬相談、在宅相談、OTC相談、LINE受付を分かりやすく分類
  3. `FOR FAMILY`：家族の薬管理や飲み合わせ相談など、相談シーン別に案内
  4. `ABOUT`：薬局の姿勢、スタッフ、会社案内を落ち着いた順序で見せる
- 視覚要素
  - 人の気配が伝わる写真、やわらかい背景色、説明しすぎない短文
  - CTAは「予約」だけでなく「相談する」という心理的ハードルの低い言葉を使う

## 4. 先にコンポーネント化する範囲

複数デザイン案を安全に試すため、以下を先に部品化する。特に薬局サイトで必須となる「営業時間」「電話」「LINE」「地図」「サービス一覧」は、どのデザイン案でも位置や見せ方だけを変えられるよう固定モジュールとして扱う。

### 4.1 レイアウト・ナビ

| コンポーネント | 役割 |
|---|---|
| `SiteHeader.astro` | ロゴ、PCナビ、モバイルドロワー、LINEボタン |
| `SiteFooter.astro` | 住所、電話、SNS、コピーライト |
| `MobileDrawer.astro` | モバイルメニュー本体。Headerから分離 |
| `FloatingLineCta.astro` | モバイル下部LINE固定ボタン |

### 4.2 セクション部品

| コンポーネント | 役割 |
|---|---|
| `SectionHeader.astro` | 英字ラベル + 日本語見出し + リード文 |
| `HeroVariant.astro` | ヒーローのパターン切り替え用ラッパー |
| `FeatureCards.astro` | 処方箋・市販薬・相談などのカード群 |
| `CtaBand.astro` | LINE・電話・アクセスなどの横長CTA |
| `InfoPanel.astro` | 営業時間・住所・注意書きなどの情報パネル |
| `VisualCard.astro` | 画像付きカード。商品・店舗紹介で共用 |

### 4.3 商品・CMS部品

| コンポーネント | 役割 |
|---|---|
| `ProductCard.astro` | 一覧の商品カード |
| `ProductBadge.astro` | カテゴリ・医薬品区分バッジ |
| `ProductHero.astro` | 個別ページ上部の商品ビジュアル + 基本情報 |
| `ProductDetailSections.astro` | 特徴・成分・用法・注意の表示 |
| `LegalNotice.astro` | 薬機法向け注意書き |

### 4.4 デザイン切り替えの土台

- `src/styles/tokens.css` を新設し、色・余白・角丸・影・フォントサイズを集約する。
- `src/styles/global.css` はリセット・共通ユーティリティ中心に整理する。
- パターンごとのCSSは以下のように分ける案を検討する。
  - `src/styles/themes/local-editorial.css`
  - `src/styles/themes/wellness-brand.css`
  - `src/styles/themes/neo-corporate.css`
- 最初はビルド時切り替えではなく、コンポーネント単位で差し替える。必要になったら `Layout` に `theme` prop を追加する。

## 7. 確定した表現方針

### 7.1 前衛度

- **本番採用候補**：高齢者にも分かりやすい範囲に留める。
  - 文字サイズ、ボタンサイズ、導線、営業時間・電話・地図の見つけやすさを最優先する。
  - 装飾やレイアウトの新しさは、情報理解を妨げない範囲で使う。
- **完全な実験案**：採用前提とは別に作成する。
  - `/design-preview/experimental-*` のように、本番候補と明確に分ける。
  - 実験案でも薬局としての必須情報（営業時間・電話・LINE・アクセス）は削らない。

### 7.2 写真・ビジュアル素材

- **初期試作は既存写真を使用する。**
  - 店舗外観、店内、商品棚、相談風景など、すでにリポジトリにある写真を優先して使う。
  - 実在感・来局前の安心感が重要な `Trust First Pharmacy` / `Community Care` は特に既存写真を中心にする。
- **生成画像・抽象図形も必要に応じて使用する。**
  - 生成画像は「実在する薬剤師・店内写真」のように誤認される使い方を避ける。
  - 抽象図形は背景、区切り、りんごモチーフ、カテゴリ表現など、情報を補助する用途に限定する。

### 7.3 動き

- **CSSアニメーションと軽量JSの両方を適切に使用する。**
  - CSS：フェード、スライド、ホバー、スクロール誘導など軽い演出に使う。
  - 軽量JS：モバイルドロワー、タブ、アコーディオン、必要最小限のスクロール連動に使う。
  - 高齢者配慮として、過度なパララックス、激しい点滅、長い待機アニメーションは避ける。
  - `prefers-reduced-motion` への配慮を実装時の確認項目に入れる。

### 7.4 フォント

- **Google Fonts の使用を許容する。**
  - ただし、読みやすさ・日本語の表示品質・読み込みコストを優先する。
  - 本文は原則として可読性の高い日本語フォントを使い、装飾的なフォントは見出し・英字ラベルに限定する。
  - 既存の `Noto Sans JP` は堅実案の基準フォントとして維持候補にする。

### 7.5 パターン別のフォント・動き・素材方針

| パターン | フォント方針 | 動き | 素材方針 | 前衛度 |
|---|---|---|---|---|
| Local Editorial | Noto Sans JP + 丸みのある見出し候補 | セクションの軽いフェード | 既存写真中心 | 低〜中 |
| Wellness Brand | Noto Sans JP + 太め英字ラベル候補 | 数値・宣言文の軽い出現 | 写真 + 抽象図形 | 中 |
| Neo Corporate | 読みやすい本文 + 強い英字見出し候補 | グリッド切替・軽いスライド | 写真切り抜き + 図形 | 中〜高 |
| Clinic Minimal | Noto Sans JP 基準 | ほぼ静的、ホバー最小限 | 既存写真を少量 | 低 |
| Ringo Pop | 丸みのある日本語フォント候補 | ふわっとしたCSS演出 | 既存写真 + りんご抽象図形 | 中 |
| Pharmacy Magazine | 本文可読性重視 + 編集的な見出し | 記事カードの軽い出現 | 既存写真 + 商品画像 | 中 |
| Trust First Pharmacy | Noto Sans JP 基準 | CTAホバー・ドロワー程度 | 外観・店内写真中心 | 低 |
| Community Care | やわらかい本文フォント候補 | 相談メニューの軽い展開 | 人の気配がある既存写真中心 | 低〜中 |
| Experimental | パターンごとに選定 | CSS + 軽量JSを広めに試す | 生成画像・抽象図形も可 | 高（本番候補と分離） |
