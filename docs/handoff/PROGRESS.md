### 商品個別ページの追加について
- **方針確定：WordPressへの移設は行わない。Astro + Headless CMSで対応する。**
- CMSは **microCMS（日本製・日本語UI）** を採用予定
  - 無料プラン（Hobby）で薬局サイト規模なら運用上問題なし
  - API呼び出し数は無制限（データ転送量に上限あるが規模的に引っかかる可能性は極めて低い）
  - クレカ登録不要でいつでも開始可能
- コンテンツ（データ）とデザイン（HTML/CSS）が完全分離するため、今後のデザイン刷新も自由に行える
- 非エンジニアがmicroCMS管理画面のUIで商品の登録・編集・公開/非公開を操作できる

### SEO対応について
- Astro（静的生成）は表示速度が速くCore Web Vitalsに有利 → SEO観点で優位
- 構造化データ（JSON-LD）・OGP・サイトマップはAstroで完全制御可能
- 記事・商品を投稿した際の自動デプロイは「GitHub + Vercel/Netlify」のCI/CDで全自動化できる
  （microCMSで保存 → 数分後に本番反映）
