# りんごちゃん薬局 プロジェクト AGENTS ルール

## 1. ドキュメントを読む順序
エージェントが作業を開始する際は、必ず以下の順序で状態を確認してください。
1. `docs/handoff/CURRENT.md` : 現在の注力タスク・ブロッカー
2. `docs/handoff/PROGRESS.md` : これまでの完了事項
3. `docs/README.md` : 迷ったときのプロジェクト全体の情報構造インデックス

## 2. 方針と制約
- **日本語出力の徹底**: 思考の出力およびファイルの記述はすべて日本語で行うこと。
- **アーキテクチャの変更**: 基本的にはVite/Astro + Vanilla CSSで構成されている。構造を変える場合はユーザーに提案・確認すること。
